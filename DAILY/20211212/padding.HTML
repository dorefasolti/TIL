기준div

영역 구분(밖에서 안으로)
1. margin : 바깥쪽 여백 영역
2. border : 테두리 영역
3. padding : 내용물과 테두리 영역 사이의 거리(안쪽여백)br 4. content : 실제 내용물이 들어있는 영역
=> 실제 요소의 크기 영역 == content + padding + border

padding

content와 border 사이의 여백을 조정하는 스타일 속성

컨텐츠영역
padding : 100px


컨텐츠영역
padding-top : 50px; padding-left-100px
