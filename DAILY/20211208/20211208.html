스크립트언어

프로그램의 동작을 사용자의 요구에 맞게 수행되도록 한다
빠르게 배우고, 짧은 소스코드로 상호작용하도록 한다


자바스크립트

웹 문서 상에 이벤트가 발생했을 경우, 추가적으로 실행해야 하는 동작을 처리 웹 브라우저에서 가장 많이 사용되는 인터프리터 방식의 스크립트언어 인터프리터방식이란, 컴파일 단계를 거치지 않고 바로 실행되는 것이다


자바스크립트 소스코드 작성 위치

1. inline(인라인)

이벤트를 부여하고자 하는 요소 내에서 소스코드를 작성
<태그명 이벤트속성="실행할소스코드">


2. internal(내부)
해당 html문서 내, script 태그 영역 안에서 소스코드를 작성


3. external(외부)
소스코드를 별도의 .js파일로 작성해서 script 태그를 가져다가 실행
외부방식은 헤드에 스크립트파일 경로를 지정해줘야 가능하다





데이터입출력

데이터입출력 관련 객체들
     1. window
    자바스크립트 내장객체로 브라우저 창이 하나 열릴 때다 자동으로 하나씩 만들어지는 객체
    브라우저 창 안에 존재하는 모든 요소들의 최상위객체이므로 생략이 가능하다
    자바에서 Object를 생각하면 된다

2. document
    웹문서마다 하나씩 만들어지는 객체로, HTML문서에 대한 정보를 가지고 있다

1. 데이터 출력 기본구문

1) window.alert('알림창에 출력할 문구');
2) window.console.log('콘솔창에 출력할 문구');
3) document.write('화면에 출력할 문구');

4) 선택한요소.innerHTML/innerText='해당요소에서출력할문구';
innerHTML과 innerText

자바스크립트에서 어떤 요소(element) 안의 값(content)을 가지고 온다거나 변경하고자 할 때 innerHTML, innerText라는 속성에 접근할 수 있다

가지고 올 때
변수 = 선택한요소.innerHTML/innerText;

변경하고자 할 때
선택한요소.innerHTML, innerText = '변경할 값';





데이터를 입력받는 기본 구문(변수에 대입 가능)

1) 변수 = window.confirm('질문내용');

confirm 호출 시 질문내용과 확인/취소 버튼이 존재하는 알림창 발생
확인버튼 클릭 시 true값, 취소 버튼 클릭 시 false를 반환
