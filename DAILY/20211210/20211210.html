해당 입력 양식의 틀
form 태그 내 submit버튼 클릭 시 해당 form 태그 내 작성된 입력값을 서버로 넘기면서 요청하는 역할을 하는 것이 form태그와 submimt버튼
form action="" method=""
*action 속성 : 해당 form 태그 안에 사용자가 입력한 값 전달하며, 요청할 서버 경로 제시(어디로 보낼 것인가?) *method속성 : 요청 전송 방식을 지정, get과 post가 있는데, get은 요청 시 사용자가 입력한 값들이 url에 노출 / post는 url에 노출되지 않음 form 태그 내 submit 버튼 클릭 시 action에 지정한 서버 경로로 method에 작성된 요청 방식을 사용해 사용자가 입력한 값들을 key = value세트로 전달
